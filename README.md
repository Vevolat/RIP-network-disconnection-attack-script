# ARP 欺骗工具 v2.0

## 简介

本工具是一个用于网络安全教学和授权测试的ARP欺骗工具，旨在帮助用户了解ARP欺骗攻击的工作原理及其潜在风险。该工具支持中英文界面，可以扫描局域网内的活跃主机并对其进行ARP欺骗攻击演示。

## 功能特性

- 自动扫描局域网内的在线主机
- 支持对单个主机进行ARP欺骗攻击
- 支持对所有在线主机进行批量ARP欺骗攻击
- 提供手动输入IP地址功能并验证主机连通性
- 安全防护机制（防止攻击网关和本机）
- 支持中英文界面切换

## 使用环境

- Windows操作系统
- Python 3.6或更高版本
- 以下Python库：
  - scapy
  - netifaces
  - ipaddress
- 管理员权限运行

## 安装依赖

```bash
pip install scapy netifaces ipaddress
```

## 使用方法

1. 以管理员权限运行命令提示符或PowerShell
2. 执行以下命令启动工具：
   ```bash
   python main.py
   ```
3. 根据提示选择界面语言
4. 在主菜单中选择相应功能：
   - 选项1：扫描并进行ARP欺骗攻击
   - 选项2：查看工具相关信息
   - 选项3：重新选择界面语言
   - 选项4：退出程序

## 注意事项

- 本工具仅用于学习和授权测试目的
- 使用本工具前请确保获得网络所有者的许可
- ARP欺骗攻击可能会导致网络连接中断
- 请在合法合规的前提下使用本工具
